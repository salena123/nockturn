# Nockturn Messenger Bot

Отдельный подпроект для мессенджер-бота студии.

## Что уже есть

- `FastAPI`-приложение
- webhook для `VK Callback API`
- базовый healthcheck
- каркас обработчиков, сервисов, клавиатур и состояния
- клиент для обращения к CRM backend

## Быстрый старт

1. Скопировать `.env.example` в `.env`
2. Заполнить токены и URL CRM
   Важно: `BOT_API_TOKEN` должен совпадать в `messenger-bot/.env` и `backend/.env`
3. Установить зависимости:

```bash
pip install -r requirements.txt
```

4. Запустить:

```bash
uvicorn app:app --reload --port 8010
```
